# MY SUDOKU

This file is a specification for a Sudoku game

## Overview
My Sudoku is a classic puzzle game where the objective is to fill a 9x9 grid with digits so that each column, each row, and each of the nine 3x3 subgrids contain all of the digits from 1 to 9 without repetition.

## Features

### Menu Items

The application contains several menu items:
- **Nouvelle Partie** - Starts a new Sudoku game with a randomly generated puzzle.
- **Réultats** - Displays the player's game statistics, including the number of games played, won, and lost.
- **Settings** - Allows the player to customize game settings such as difficulty level, timer options, and sound effects.
- **Profile** - Enables the player to create and manage their user profile, including username and avatar.

### New Game

When the player selects "Nouvelle Partie," a new Sudoku puzzle is generated based on the selected difficulty level (Easy, Medium, Hard). The grid is displayed with some cells pre-filled, and the player can start filling in the empty cells with its keyboard or mouse.

### Résults

The "Réultats" section provides a summary of the player's performance, including:
- Total games played
- Total games won

### Settings
In the "Settings" menu, players can adjust various options:
- Theme selection (light or dark mode)


### Profile
The "Profile" section allows players to:
- Update their username
- Change their avatar
